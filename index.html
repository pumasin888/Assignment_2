Task 1: Installation of Arduino IDE software on an Ubuntu VM.

1.1 Install the Arduino IDE on the Ubuntu VM.

We install Arduino using App center. And the version that we install is 1.8.19 because it is the lastest version.

1.2 Open the Arduino IDE and open the Blink sketch from the Arduino Examples.

Now we will test our arduino by using the Blink sketch example from arduino.

1.3 Select an Arduino board as the target, such as Arduino Uno or Nano.

Now we need to specify the arduino board that we use the board that I used is Arduino Uno. Then we have to specific the port that we connected also.

1.4 Build the Arduino sketch example.

This step is to build and verify the code that we will upload to our arduino. You can click the verify button above to build our code.

1.6 Upload the compiled Arduino sketch to the Arduino board.

Now this is the last step. It is when we upload our code to the arduino board. Simply just click the upload button the button that look like arrow that pointing to the right. Now if nothing wrong our arduino board show have some orange LED that are blinking !



Task 2: Installation of VS Code IDE software and extensions

2.1 Install VS Code IDE for Windows (Host OS).

Download vscode for remote to our vm ware for coding our adruino board.

2.2 Install the Remote Development Extension Pack for VS Code IDE.

Download the extension for remoting the vmware.

2.3 Connect VS Code to the remote machine (Ubuntu).

First we need to config our vmware network to use bridge network to allow it to make some lan connecting.

First we have to create our ssh connection to connected to our vmware.

Now we are connected we should see that our terminal change to the vmware already.

2.4 Install the PlatformIO extension for Remote Development.

The next step is to download the PlatformIO extension it allow us to write some arduino code in vscode.

2.5 Create a project using PlatformIO and the Arduino framework

Now we have to create our project with the platformIO by use the command `pio project init --board uno`

If it is success we should see that it create folder for us. If we want to sketch some arduino code go to the src folder and now you can create some .cpp file that will be the arduino code.

2.6 Build the Arduino sketch and upload it to the target.

Same as task 1 when we finish writting some code we have to build and verify it by cliking the build button below. It should say SUCCESS in the terminal. Now we ready to upload it.
